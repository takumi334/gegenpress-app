# 戦術ボード機能

## 利用可能なスレッド

戦術ボードは **試合前スレ（PRE_MATCH）** と **試合中スレ（LIVE_MATCH）** でのみ利用できます。

- **PRE_MATCH**: 試合前の予想配置・狙いどころを書ける。ボード機能をフルで利用可能（自由線・色・太さ・コメントは今後実装）。
- **LIVE_MATCH**: 試合中の気づきを素早く共有する用途。現状は PRE_MATCH と同じ UI。将来 LIVE_MATCH 用の軽量版 UI に切り替えやすい構造にしてある。

## 利用不可のスレッド

- **POST_MATCH（試合後スレ）**: 戦術ボードの**新規投稿は不可**。既に投稿済みの戦術ボードの**閲覧のみ可能**。
- **GENERAL（通常・雑談スレ）**: 戦術ボードの表示・投稿は**不可**。

## アクセス制御

- 戦術ボード作成ページに直接 URL でアクセスした場合も、スレッド種別をチェックする。
- PRE_MATCH / LIVE_MATCH 以外のスレッドでは作成不可とし、**スレッド詳細へリダイレクト**し、エラーメッセージ「このスレッドでは戦術ボードの新規投稿はできません」を表示する。
- 保存 API でもサーバー側でスレッド種別を検証し、許可されていない場合は 403 を返す。

## スレッド種別（threadType）

| 値 | 説明 |
|----|------|
| PRE_MATCH | 試合前スレ |
| LIVE_MATCH | 試合中スレ |
| POST_MATCH | 試合後スレ |
| GENERAL | 通常・雑談スレ |

旧値（lineup, halftime, postmatch, prematch）はコード内で上記に正規化される。

## 将来の拡張

- **LIVE_MATCH 用軽量版 UI**: コンポーネントは `variant="full" | "compact"` で出し分け可能な設計。LIVE_MATCH では `compact` にし、素早くメモだけ投稿する軽量版に発展できる。
- 自由線・色変更・太さ変更・コメントは今後の実装予定。
